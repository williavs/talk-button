Download Voice Prompt

Voice Prompt is a desktop application that enables voice-to-text transcription with a modern, dark-themed interface.

Download for Mac
Note: Since this app isn't signed with Apple, you'll need to follow the instructions below to open it.

Installation Instructions:

  1. Download the ZIP file using the button above
  2. Double-click the ZIP file to extract it
  3. Drag "Voice Prompt.app" to your Applications folder
  4. The first time you open the app:
    • Right-click (or Control-click) on the app
    • Select "Open" from the menu
    • Click "Open" in the security dialog
  5. If needed, go to System Preferences > Security & Privacy > General and click "Open Anyway"

System Requirements:

  • macOS 10.15 or later
  • Microphone access (you'll be prompted to allow this)
  • Internet connection for transcription
